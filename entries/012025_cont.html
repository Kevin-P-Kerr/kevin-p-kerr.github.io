Closest Rational Approximations of Square Roots

Early, I asked what is the algorithm for determining the closest rational approximation to a square root, given the numerator/denominator are two n-digit/bit numbers?


Obviously, one way to answer this question is iterate through every n-bit/digit rational number and, by taking their absolute difference with the square of the number we're seeking, we can determine which most closely approximates to it. This is fairly simple to code but also very cleary ineffecient. For if we wish to find every n-bit rational number, then there are `2^n x 2^n=2^2n` of them. But even if we only iterate over a subset, there are still many over which to iterate. Additionally, the linked code is flawed--when finding the difference between two rational numbers we don't reduce by the gcd of the numerator and denominator, and so we quickly run into issues with integer overflow.
